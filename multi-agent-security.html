Security Analysis: AI Guard Redundancy

This model evaluates the probability of a Critical System Failure ($p^*$) where a Work Agent hallucinates malicious code and no Guard Agent detects it.

Mathematical Derivation

1. Probability at least one of $n$ Work Agents is malicious: $P_M = 1 - (1 - p)^n$.

2. Probability $k$ Guards all fail to detect it: $P_E = (1 - q)^k$.

3. Total system failure probability: $$p^* = [1 - (1 - p)^n] \cdot (1 - q)^k$$

4. Required Guards $k$ to ensure $p^* < p_t$: $$k > \frac{\ln(p_t) - \ln(1 - (1 - p)^n)}{\ln(1 - q)}$$
