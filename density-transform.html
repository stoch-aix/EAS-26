Density Transformation with Explicit Domains

In Computer Science simulations (e.g., modeling non-linear sensor data), we transform a source variable $X$ into a target variable $Y$. Defining the support (the set of values where the density is non-zero) is essential.

1. Source Distribution: $X \sim U(0,1)$ $$f_X(x) = \begin{cases} 1 & x \in [0, 1] \\ 0 & \text{otherwise} \end{cases}$$
2. Transformation: $Y = g(X) = X^2$. Since $x \in [0, 1]$, the domain of $Y$ is $y \in [0, 1]$ (for $y > 0$).
3. Density Formula: For $y \in (0, 1]$: $$f_Y(y) = f_X(g^{-1}(y)) \cdot \left| \frac{d}{dy} g^{-1}(y) \right| = 1 \cdot \frac{1}{2\sqrt{y}} = \frac{1}{2\sqrt{y}}$$

Input $f_X(x)$ for $x \in [0,1]$

Output $f_Y(y)$ for $y > 0$
